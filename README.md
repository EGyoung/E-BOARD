# E-Board

预览地址: https://egyoung.github.io/E-BOARD

### 开发流程

1. 克隆项目后，运行 `pnpm install` 安装依赖
2. 运行 `pnpm dev` 启动开发服务器
3. 访问 `http://localhost:3001` 查看项目
